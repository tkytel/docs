              IP over Private LoRa に関する予備実験の報告

1. はじめに

    [TC36] に引き続いて、Private LoRa を用いて ICMP パケットを伝送する
  予備実験を行ったので、これについて簡単に報告する。

    なお、本文書では、断りなく [TC36] で用いられている用語を用いる。ま
  た、MCU と通信モジュールとを単一の筐体に収めたものを通信機という。

2. 使用機材

    概ね [TC36] で述べられる機材に同じである。ただし、一方の通信機にお
  いては、アンテナとして秋月電子通商で販売されている TX4G-XPFC-300（販
  売コード: 130288）[販売コード 130288] を用いる。

3. フレームの構造

    [TC36] で述べられるフレームの構造に同じである。ただし、ARP を用い
  ていない。そのため、フレームのヘッダ部に含まれる Destination Device
  Address や Source Device Address には具体的な値を指定せず、いづれも
  ブロードキャストアドレス（0xFFFF）を指定している。

4. 通信モジュールのパラメータ

    概ね [TC36] で述べられる通信モジュールのパラメータに同じである。た
  だし、伝送速度を最大とするため、拡大係数及び帯域幅をそれぞれ 5 及び
  500 kHz に変更している。

5. 通信実験

    一方の通信機はコンピュータに接続され、コンピュータ上ではネットワー
  クデバイスとして振る舞うようにシステムを構成した。以降、これを移動機
  という。他方の通信機の MCU はプログラムを書き換えられ、スタンドアロ
  ンで ICMP Echo Request Message を受け、ICMP Echo Reply Message を応
  答できるようにシステムを構成した。以降、これを固定機という。なお、固
  定機のシステムには誤りがあり、ICMP パケットのチェックサムを正しく計
  算できなかった。しかし、応答パケットの到達の有無を確認する上では、問
  題なく動作した。

    固定機のアンテナとして TX4G-XPFC-300 を用い、これを宍道湖大橋 [1]
  に設置した。アンテナは橋の手摺りに固定された。宍道湖周では、移動機か
  ら ICMP Echo Request Message を送信し、ICMP Echo Reply Message の受
  信の可否を確認した。

    2026 年 4 月 25 日 23 時 40 分ごろから同月 26 日 2 時 20 分ごろに
  かけて実験を行った。実験を行った時点における、松江市の天気は晴れであ
  り、出雲市においては若干の霧を認めた。

    通信の可否について、下表に示す。

  |               地点（付近）             |   距離  | 見通し | 通信 |
  | -------------------------------------- | ------- | ------ | ---- |
  | 朝日の見える駐車場 [2]                 |  0.8 km |   有   |  可  |
  | 島根県運転免許センター駐車場 [3]       |  6.5 km |   無   | 不可 |
  | 道の駅 秋鹿なぎさ公園 [4]              |  9.2 km |   無   | 不可 |
  | ローソン・ポプラ 平田一畑店 [5]        | 15.2 km |   無   | 不可 |
  | 宍道湖西岸なぎさ公園 [6]               | 16.3 km |   有   | 不可 |
  | 日本シジミ研究所 [7]                   |  7.5 km |   無   | 不可 |
  | セブン-イレブン 松江玉湯町布志名店 [8] |  4.5 km |   有   |  可  |

    通信速度が最大となるようなパラメータを用いた場合、見通しのある環境
  において、約 4.5 km の離れた二点間で通信を実現できることを確認した。

    見通しのある環境において、より距離のある二点間で通信できるのか、確
  認する必要がある。また、通信速度を低下させることにより、到達距離を大
  きくできると考えられるため、これについても検証する必要がある。

6. 費用

    本実験に係る費用は、[TC36] で既に述べられたものに加え、固定機に用
  いた TX4G-XPFC-300 による 1540 円である。なお、移動のための燃料費は
  計上を省略する。

7. おわりに

    「大丈夫、きっとうまくいく」と信じ込んでいる。
